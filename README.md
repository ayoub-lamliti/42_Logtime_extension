# ⏱ Logtime Companion

A lightweight Chrome Extension strictly for 42 Network students to track their weekly logtime and get milestone notifications (20h, 25h, and 30h).

## 🚀 How to Install (Chrome / Brave)

Since this extension is currently in beta and not yet published on the Chrome Web Store, you need to install it manually. Follow these simple steps:

### Step 1: Download the Extension
1. Download the provided `logtime-companion.zip` file (or the extension folder).
2. Extract (unzip) the folder to a safe place on your computer (e.g., your `Documents` folder). 
*⚠️ Important: Do not delete or move this folder after installation, otherwise the extension will stop working.*

### Step 2: Enable Developer Mode
1. Open your browser and type `chrome://extensions/` in the address bar (use `brave://extensions/` if you are on Brave).
2. In the top-right corner of the page, toggle on **Developer mode**.

### Step 3: Install the Extension
1. Click the **Load unpacked** button that just appeared in the top-left corner.
2. Select the extracted folder you prepared in Step 1. *(Make sure you select the specific folder that contains the `manifest.json` file).*
3. The Logtime Companion card should now appear in your list of extensions.

### Step 4: Pin and Connect
1. Click the puzzle icon 🧩 in the top-right of your browser's toolbar.
2. Click the pin icon 📌 next to **Logtime Companion** to keep it visible.
3. Click the extension icon, select which day your week starts on, and click **Connect with 42**.
